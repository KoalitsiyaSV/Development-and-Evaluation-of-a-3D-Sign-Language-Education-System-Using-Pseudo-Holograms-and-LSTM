## 3. 기능
립모션을 통해 데이터를 입력받고 이를 서버로 전송하여 수화를 인식합니다.
인식한 수화를 다시 클라이언트로 전송합니다.
클라이언트는 학습과 테스트 2개의 기능을 가집니다.
